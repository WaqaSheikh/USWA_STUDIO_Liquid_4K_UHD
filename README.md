## 📂 How to Use
1. Clone the repository or download the ZIP.
2. Ensure all thumbnail images are inside the `thumbnails/` folder.
3. Open `index.html` in any modern browser.
4. Click on any thumbnail to view the corresponding 4K video preview via YouTube.
